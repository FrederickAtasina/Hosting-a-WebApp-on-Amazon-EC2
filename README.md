# EC2 Web Application Deployment

## Overview
This project demonstrates how to deploy a Linux-based web server on AWS using Amazon EC2.

## Architecture
- EC2 instance in a public subnet
- Security Group controlling SSH and HTTP access

## Services Used
- Amazon EC2
- Amazon VPC
- Security Groups

## Security
- SSH access restricted to my IP
- Least-privilege inbound rules

## Cost Control
- Free Tier eligible instance
- Resources terminated after testing

## Lessons Learned
- Linux server setup
- AWS networking basics
- Cloud security best practices
